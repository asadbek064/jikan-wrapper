## About
jikan-wrapper-ts is a wrapper for Jikan v4 API.

## Installation
```sh-session
npm install jikan-wrapper
```

## Example 
```js
// import
const jikan = require('jikan-wrapper');

// usage example
getAnimeById(5114)
  .then(result => {
    console.log(result);
  });
```
